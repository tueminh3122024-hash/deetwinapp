DeeTwin - DATA INPUT SYSTEM

1. No-device mode (Mặc định) MVP

Nếu bạn chưa có thiết bị đeo, DeeTwin cung cấp hệ thống Synthetic Baseline để mô phỏng hoạt động cơ thể. Bạn sử dụng Quick Input Hub để nhập thủ công:

  • Giấc ngủ: Đánh giá thời gian phục hồi.
  • Stress: Căng thẳng tâm lý ảnh hưởng trực tiếp đến đường huyết.
  • Meal: Ghi chú bữa ăn để AI soi nguyên nhân tăng vọt.

2. Apple Health / Health Connect

Kết nối trực tiếp với Apple Watch hoặc iPhone HealthKit để lấy dữ liệu tự động:

  • HR / HRV: Nhịp tim thời gian thực.
  • Sleep / Steps: Theo dõi chuyển động và phục hồi.

3. CGM Integration (Cảm biến đường huyết)

Manual: Nhập chỉ số thử máu hoặc quét ảnh cảm biến (OCR Ready).

Auto: Đồng bộ API từ Dexcom hoặc Nightscout Cloud để Dashboard nhảy số mỗi 5 phút.

Cốt lõi: Công thức OSAP Brain

AI Gemini sẽ đọc chuỗi 7 Metrics + các thông số trên để sinh ra OSAP (Operational State Action Plan):

- MSI (Stability): Cho biết bạn đang ổn định hay biến thiên.

- EIB (Energy): Cho biết bạn còn bao nhiêu pin năng lượng.

- MGC (Control): Đánh giá kỷ luật ăn uống của bạn.
